# spring_aws
> Spring framework, AWS ec2서버 배포, RDS 연동

*현재 링크 닫힘* 

## 프로젝트 개요
* 개발 언어: JAVA (JDK 1.8)
* 프레임워크: Spring Framework 4.5.1
* DB: MySQL 8.0
* 서버: AWS EC2
* 빌드 도구: Maven
* 테스트: JUnit 4.13
